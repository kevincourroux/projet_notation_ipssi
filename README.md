# projet_notation_ipssi

## Installation

```bash
git clone https://github.com/kevincourroux/projet_notation_ipssi.git
```

## Usage

### Start project Back

```bash
cd back/
docker-compose up
```
### Start project Front

```bash
cd front/angular/
docker-compose up
```

Open a browser and go to [localhost:3001](http://localhost:3001/)
